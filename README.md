# Jarvis: Voice-Activated Desktop Assistant 🤖🔥

A responsive, local Python-based voice assistant inspired by Iron Man's Jarvis. This project handles real-time microphone input, processes text commands, automates web activities, and utilizes modern API frameworks to deliver voice-driven responses.

## 🚀 Key Features
* **Speech-to-Text Recognition:** Leverages the `speech_recognition` library along with Google's speech endpoints to accurately parse voice data.
* **Text-to-Speech Engine:** Integrated with `pyttsx3` and `gTTS` for seamless, natural voice delivery.
* **Intelligent Querying:** Connected to modern generative AI APIs (`google-genai`) to answer complex queries conversationally.
* **Live Automated Navigation:** Supports instant automated shortcuts for opening critical web portals like Google and Facebook.
* **Live News Streaming:** Pulls top breaking news headlines instantly using live API requests (`requests`).

## 🛠️ Tech Stack & Libraries
* **Language:** Python 3.12+
* **Core Modules:** `speech_recognition`, `pyaudio`, `pyttsx3`, `gTTS`, `requests`, `pygame`, `google-genai`
* **Environment Management:** Isolated virtual environments (`.venv`)
